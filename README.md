# Reef IT Helpdesk Dashboard

Live metrics dashboard for Reef Capital Partners IT Helpdesk.

## Overview

This dashboard displays real-time statistics from our Jira Service Desk:
- 14-day ticket creation and resolution trends
- Current ticket status breakdown
- Workload indicator

## Data Updates

- **Business hours (8am-5pm MT, Mon-Fri):** Every 30 minutes
- **After hours:** Every 4 hours

Data is automatically refreshed by Zara AI.

## Technology

- Static HTML/CSS/JS dashboard
- Chart.js for visualizations
- GitHub Pages for hosting
- Automated via cron jobs
